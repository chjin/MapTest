UID : 프로세스를 실행한 ID
PID : 프로세스 번호
PPID : 부모 프로세스 번호
C : CPU 사용량(%값)
STIME : 프로세스 시작 날짜,시간
TTY : 프로세스가 실행된 터미널의 종류와 번호
TIME : 프로세스 실행 시간
CMD : 실행되고 있는 프로그램 이름(명령)
STAT
 - R : 실행중(running)
 - S : 인터럽트가 가능한 대기(sleep) 상태
 - T : 작업 제어에 의해 정지된(stopped) 상태
 - Z : 좀비 프로세스(defunct)
 - s : 세션 리더 프로세스 
 - + : 포그라운 프로세스 그룹 
 - l(L) : 멀티 스레드
USER : 사용자 계정 이름
%CPU : CPU 사용량을 퍼센트로 표시
%MEM : 물리적 메모리 사용량을 퍼센트로 표시
VSZ : 사용하고 있는 가상 메모리 크기(KB)
RSS : 사용하고 있는 물리적 메모리 크기(KB)
START : 프로세스 시작 시간
